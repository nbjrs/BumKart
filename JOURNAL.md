# May 24: Mahoraga wheel V1 made!

As promised in my project description, I am going to make the steering wheel for the kart mahoragas wheel from JJK. To make this I researched the shape and designed one that can work on onshape. It should be able to be assembled through 3d printing the balls and the rim, while cutting 20x20 mm wooden square beems the right size and painting them gold to match. The lapse link is posted below for the full 53 minute work session. My modelling was a little sketchy but also I want it to work IRL so I had to do some thinking on how feasible it is. Currently planning on working on the actual BOM more in depth and starting chassis design later tdy!
